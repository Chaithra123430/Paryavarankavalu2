Paryavaran-Kavalu
Android App Development using GenAI – Environmental Monitoring

Application
Paryavaran-Kavalu is an Android-based environmental monitoring application developed to identify, report,
and manage illegal waste dumping locations using geo-tagged map locations and community participation.
The application helps users report waste blackspots, categorize waste types, monitor cleanup activities, and
earn Eco-Karma reward points for contributing to environmental cleanliness.
 
 Features
 Google Maps integration for geo-tagged waste reporting
 Real-time GPS location tracking
 Waste category selection
 Report illegal waste dumping spots
 Mark waste spots as cleaned
 Eco-Karma reward points system
 User impact dashboard
 Marker-based waste visualization on map
 Simple and user-friendly Android UI

 App Workflow
User opens the application.
Current location is fetched using GPS.
Google Maps displays the user location.
User clicks on “New Report”.
User selects a waste category.
A waste marker is placed on the map.
Users can confirm cleanup status.
Eco-Karma points are updated.

Problem Statement
Illegal garbage dumping and unmanaged waste disposal in public areas create serious environmental and
public health issues. Local volunteers and cleaning authorities often struggle to identify waste blackspots
due to the absence of a centralized reporting and tracking system.
Paryavaran-Kavalu addresses this issue by providing a geo-tagged Android application that enables users to
report waste locations, categorize waste types, and support community-based cleanup activities efficiently.
